# Comandos do Projeto

## Desenvolvimento
- `npm run dev`: iniciar servidor de desenvolvimento com Turbopack.
- `npm run start`: iniciar servidor em produção.

## Build
- `npm run build`: gerar build otimizado.

## Qualidade
- `npm run lint`: analisar com ESLint.
- `npm run type-check`: `tsc --noEmit`.

## Testes
- `npm run test:unit`: Vitest com ambiente `jsdom`.
- `npm run test:e2e`: Playwright.
- `npm run test`: unitários e e2e.

## Utilitários
- `npm run analyze`: opcional para analisar bundle quando disponível.
