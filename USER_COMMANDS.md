# Guia para Usuários (Time de Negócio)

## Como rodar
- `npm run dev` e acessar `http://localhost:3000`.

## Como validar
- Conferir fluxos principais (lead, vendas) em mobile e desktop.
- Verificar mensagens e feedbacks (erros/sucesso).

## Como testar
- Executar `npm run test:unit` e revisar resultados.

## Como reportar
- Abrir issue com descrição, passos, evidências (prints) e prioridade.

## Checklist
- Funcionalidade atende objetivo?
- UX clara e responsiva?
- Mensagens adequadas?
- Sem erros aparentes?
